# Database Automation

Databases are stateful, making them harder to manage with Terraform than stateless web servers.

## The Separation of Concerns

* **Terraform:** Provision the infrastructure (EC2, EBS, Security Groups).
* **Ansible:** Configure the OS and Database Software (users, conf files).
* **State Management:** Never store secrets in your `tfstate`. Use AWS Secrets Manager.
