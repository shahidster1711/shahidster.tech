# Blog Images Directory

This directory contains images for blog posts.

## Usage

Place blog post featured images here and reference them in your Markdown frontmatter:

```markdown
---
title: "My Blog Post"
image: "/blog-images/my-featured-image.jpg"
---
```

## Recommendations

- Use descriptive filenames
- Optimize images for web (recommended < 200KB)
- Aspect ratio: 16:9 for featured images
- Supported formats: JPG, PNG, WebP
