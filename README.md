## üíª Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/shahidster1711/shahidster.tech.git
   cd shahidster.tech
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.[5][3]

***

## üì¶ Building for Production

```bash
npm run build
```
